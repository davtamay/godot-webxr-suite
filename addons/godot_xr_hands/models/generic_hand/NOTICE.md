# generic-hand models

`left.glb`, `right.glb`, and `profile.json` come from the W3C Immersive Web
Community Group's WebXR Input Profiles assets package:

- Source: https://github.com/immersive-web/webxr-input-profiles (packages/assets)
- Distribution: `@webxr-input-profiles/assets` (npm), profile id `generic-hand`
- License: MIT, Copyright (c) 2019 Amazon — see `LICENSE.md` in this directory

The meshes are rigged with the 25 standard WebXR hand joints (flat sibling
skeleton; each bone is posed directly in tracker space). They depict a generic
human hand — no manufacturer trademark is involved.
